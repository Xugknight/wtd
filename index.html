Tips

  • Save stores to your browser (autosaves as you type).
  • .txt (Ctrl+S) downloads plain text.
  • .html (Ctrl+Shift+S) exports with formatting.
  • Open… loads a local .txt/.md/.html.
  • Paste as text strips formatting; toggle off to keep rich paste.
  • Monospace uses a code font that aligns characters—great for code or tables.
  • Wrap keeps long lines visible without horizontal scrolling. Turn off for exact formatting.
  • MD Preview shows a live Markdown view of the pad’s content.
  • Tabs: Click names to switch, Ctrl/Cmd-Click to rename, Right-click to delete, ＋ to add.
